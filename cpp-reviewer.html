What is C++?

C++ is a general-purpose programming language that supports object-oriented, procedural, and generic programming. It's widely used for system software, game development, and performance-critical applications.

Object-Oriented Programming

Use classes, inheritance, and polymorphism for modular code design.

Memory Management

Control memory allocation and deallocation with pointers and smart pointers.

Standard Template Library

Utilize powerful data structures and algorithms from the STL.

Performance Optimization

Write efficient code with low-level control and optimization techniques.

Basic C++ Functions

1. Function Definition

Define functions with return types, parameters, and function bodies.

int add(int a, int b) {
    return a + b;
}
int main() {
    std::cout << add(5, 3) << std::endl;
    return 0;
}

2. Class and Objects

Create classes with member functions and instantiate objects.

class Rectangle {
public:
    int width, height;
    int area() {
        return width * height;
    }
};
int main() {
    Rectangle rect;
    rect.width = 10;
    rect.height = 5;
    std::cout << rect.area() << std::endl;
}

3. Pointers

Use pointers for memory manipulation and dynamic allocation.

int* ptr = new int(42);
std::cout << *ptr << std::endl;
delete ptr;

4. Vectors

Use std::vector for dynamic arrays with automatic memory management.

#include <vector>
std::vector<int> numbers = {1, 2, 3, 4, 5};
numbers.push_back(6);
for(int num : numbers) {
    std::cout << num << " ";
}

5. File I/O

Read from and write to files using fstream.

#include <fstream>
std::ofstream file("output.txt");
file << "Hello, World!" << std::endl;
file.close();

6. Exception Handling

Use try-catch blocks to handle runtime errors gracefully.

try {
    int result = 10 / 0;
} catch (const std::exception& e) {
    std::cout << "Error: " << e.what() << std::endl;
}
