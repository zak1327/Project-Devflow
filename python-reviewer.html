What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used for web development, data analysis, AI, and automation.

Write clean, readable code with minimal boilerplate.

Basic Python Functions

1. Function Definition

Define functions using the def keyword with parameters and return values.

def greet(name):
    return f"Hello, {name}!"

print(greet("World"))

2. List Operations

Use lists for mutable sequences with various built-in methods.

numbers = [1, 2, 3, 4, 5]
numbers.append(6)
squared = [x**2 for x in numbers]
print(squared)

3. Dictionary Usage

Create key-value pairs with dictionaries for efficient data lookup.

person = {"name": "Alice", "age": 30}
person["city"] = "New York"
print(person["name"])

4. File Handling

Read from and write to files using context managers.

with open("example.txt", "w") as file:
    file.write("Hello, World!")

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

5. Classes and Objects

Define classes with methods and create instances (objects).

class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        return f"{self.name} says woof!"

dog = Dog("Buddy")
print(dog.bark())

6. Exception Handling

Use try-except blocks to handle errors gracefully.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
finally:
    print("Execution completed.")
