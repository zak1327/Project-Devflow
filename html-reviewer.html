What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags.

Semantic Structure

Use semantic elements like header, nav, main, and footer for better accessibility and SEO.

Forms and Input

Create interactive forms with various input types for user data collection.

Multimedia

Embed images, videos, and audio content using appropriate HTML elements.

Links and Navigation

Create hyperlinks and navigation structures to connect web pages.

Basic HTML Elements

1. Headings

Define headings with h1 to h6 tags for document structure and hierarchy.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

2. Paragraphs

Use p tags to define paragraphs of text content.

<p>This is a paragraph of text.</p>
<p>Another paragraph here.</p>

3. Links

Create hyperlinks using the a tag to navigate between pages.

<a href="https://example.com">Visit Example</a>

4. Images

Embed images using the img tag with src and alt attributes.

<img src="image.jpg" alt="Description of image">

5. Lists

Create ordered (ol) and unordered (ul) lists with li elements.

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

6. Forms

Use form elements to collect user input with input, textarea, and button.

<form>
  <input type="text" name="username">
  <button type="submit">Submit</button>
</form>
