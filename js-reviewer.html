What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It can update and change both HTML and CSS, and can calculate, manipulate, and validate data.

DOM Manipulation

Interact with HTML elements and modify the document structure dynamically.

Event Handling

Respond to user actions like clicks, key presses, and form submissions.

Data Processing

Work with arrays, objects, and perform calculations on user data.

Asynchronous Programming

Handle operations that take time, like API calls, without blocking the UI.

Basic JavaScript Functions

1. Function Declaration

Define reusable blocks of code using function declarations.

function greet(name) {
    return `Hello, ${name}!`;
}
console.log(greet('World'));

2. Arrow Functions

Use concise syntax for function expressions with arrow functions.

const add = (a, b) => a + b;
console.log(add(5, 3)); // 8

3. Array Methods

Use built-in methods like map, filter, and reduce for array operations.

const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

4. DOM Selection

Select HTML elements using methods like getElementById and querySelector.

const element = document.getElementById('myDiv');
element.textContent = 'Hello, DOM!';

5. Event Listeners

Attach event handlers to elements to respond to user interactions.

const button = document.querySelector('button');
button.addEventListener('click', () => {
    alert('Button clicked!');
});

6. Promises

Handle asynchronous operations using Promises for better code flow.

const fetchData = () => {
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve('Data loaded'), 1000);
    });
};
fetchData().then(data => console.log(data));
