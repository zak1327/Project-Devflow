Key Programming Functions

1. Function Declaration

Defines a reusable block of code that performs a specific task.

function greet(name) {
    return "Hello, " + name + "!";
}
// Usage:
greet("Alice"); // "Hello, Alice!"

2. Arrow Function

A concise way to write functions using the => syntax.

const add = (a, b) => a + b;
// Usage:
add(2, 3); // 5

3. Array Map

Creates a new array by applying a function to each element of an array.

const numbers = [1, 2, 3];
const doubled = numbers.map(n => n * 2);
// doubled: [2, 4, 6]

4. Filtering Arrays

Returns a new array with elements that pass a test function.

const numbers = [1, 2, 3, 4];
const even = numbers.filter(n => n % 2 === 0);
// even: [2, 4]

5. Object Destructuring

Extracts properties from objects into variables.

const user = { name: "Bob", age: 25 };
const { name, age } = user;
// name: "Bob", age: 25
